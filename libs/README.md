# Libraries

Библиотеки проекта

## crsf/

- `CrsfSerial.cpp` - Реализация CRSF протокола
- `CrsfSerial.h` - Интерфейс CRSF
- `crsf_protocol.h` - Определения протокола

- PWM управление
- Таймеры
- Задержки

## SerialPort.cpp

Обертка для работы с последовательными портами
